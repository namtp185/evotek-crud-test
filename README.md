# evotek-crud-test

Chương trình demo quản lý user.


## Cài đặt thư viện

```
npm install
```

## Chạy chương trình

Khởi chạy webserver 
```
npm start
```
Đăng nhập với vai trò là Admin User

```
username: "admin"
password: "admin"
```

Hoặc Normal User

```
username: "user"
password: "user"
```


Mở địa chỉ [`localhost:3000/`](http://localhost:3000/) trên trình duyệt.
